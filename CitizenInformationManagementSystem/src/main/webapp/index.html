About the Site

The Website was primarily developed to have a computerized and well-organized Citizen Information Management System. Here are the 3 main user Login types :

  • Admin :

    Has the right to not only view and search any citizen from the database but also can manage them, for eg. Edit the Citizen details, add-delete official-viewer accounts and has full privelege to the databases.

  • Official Viewers :

    An Official-Viewer can view any Citizen's details but will not be able to make any changes to them. eg. for this type of account holders are Police, Hospitals, government offices etc.

  • Citizen :

    General public can login to this account. If they want to view the details of any other citizens, then they must know their citizen ID and the view password related to that specific(destination) account.
